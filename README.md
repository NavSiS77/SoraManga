# SoraManga

Дипломный SPA-проект: платформа чтения манги на **Vue 3**, **Vue Router**, **Pinia**.

## Установка

```bash
npm install
```

### Разработка (hot-reload)

```bash
npm run serve
```

### Сборка для production

```bash
npm run build
```

### Линтинг

```bash
npm run lint
```

### Настройка Vue CLI

См. [Configuration Reference](https://cli.vuejs.org/config/).
